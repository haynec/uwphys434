The plot is not symmetric as the margins for 1 sigma are different above and below the mean value. Since the observed value is not equally likely to be stronger or weaker the distribution is biased.
function [thresh] = sigthresh(x,mean_val,sig)
poisson = makedist("Poisson", 'lambda',mean_val);
prob_sig = 1-1/(1/(1-erf(sig/(sqrt(2)))));
y_vals = pdf(poisson,x);
for i = 1:length(x)
check = sum(y_vals(1:i));
if check > prob_sig
thresh = i;
break
end
end
end
